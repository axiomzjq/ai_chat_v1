# RAG_DOCS — 智能体方法论文档库

## 目录作用

`RAG_DOCS/` 存放支撑 AI 智能体运行的原始方法论文档，是系统提示词 (`src/lib/prompts.ts`) 的知识来源。所有文档已提取文本并上传至智谱AI向量知识库，AI 在运行时可实时检索补充细节。

## 使用方式

1. **本地提示词**：`src/lib/prompts.ts` 内嵌了各板块的核心框架与规则边界，确保 AI 不偏离主线。
2. **知识库查询**：详细技法（具体追问话术、完整字段定义、拍摄参数细则等）由 AI 在对话中通过智谱向量知识库实时检索补充。
3. **文档更新**：如需更新方法论，修改对应 docx 后重新上传至智谱知识库，并视情况更新 `src/lib/prompts.ts` 中的核心框架。

## 注意事项

- 解压后的 `.docx`、`.png`、`.mmd` 等文件已纳入 Git，作为代码仓库的参考资产。
